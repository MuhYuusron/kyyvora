PANDUAN MENAMBAHKAN GAMBAR
============================

Folder ini membutuhkan 2 file gambar:

1. kyyvora.png
   - Logo untuk navbar (hanya logo, tanpa teks)
   - Ukuran: 200x200px - 500x500px (tinggi akan disesuaikan ke 60px)
   - Format: PNG dengan background transparan (RECOMMENDED)
   - Letakkan file di: public/foto/kyyvora.png
   
   PENTING: 
   - Gunakan background transparan agar logo terlihat bagus
   - Logo akan ditampilkan di navbar dengan background putih
   - Pastikan logo memiliki kontras yang baik dengan background putih

2. tentang.png
   - Gambar untuk section "Tentang Kyyvora"
   - Ukuran: 800x600px - 1200x900px
   - Format: PNG atau JPG
   - Letakkan file di: public/foto/tentang.png

CARA MENAMBAHKAN:
1. Siapkan file gambar dengan nama yang sesuai
2. Copy file ke folder public/foto/ ini
3. Restart development server jika sedang berjalan:
   - Stop server (Ctrl+C)
   - Jalankan lagi: npm start
4. Refresh browser (Ctrl+F5)

PERUBAHAN TERBARU:
- Navbar sekarang menggunakan background PUTIH (#fff)
- Logo akan terlihat lebih jelas dengan background putih
- Tulisan "KYYVORA" sudah dihapus, hanya logo saja
- Logo memiliki drop shadow untuk efek depth

TIPS:
- Untuk logo, gunakan warna yang kontras dengan putih
- Jika logo Anda berwarna terang, tambahkan outline/stroke
- Test logo di berbagai ukuran layar
- Compress gambar untuk loading cepat

Untuk panduan lengkap, baca: IMAGE_GUIDE.md di root folder

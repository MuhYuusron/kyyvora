PANDUAN MENAMBAHKAN GAMBAR
============================

Folder ini membutuhkan 2 file gambar:

1. kyyvora.png
   - Logo untuk navbar
   - Ukuran: 200x200px - 500x500px
   - Format: PNG dengan background transparan
   - Letakkan file di: public/kyyvora.png

2. tentang.png
   - Gambar untuk section "Tentang Kyyvora"
   - Ukuran: 800x600px - 1200x900px
   - Format: PNG atau JPG
   - Letakkan file di: public/tentang.png

CARA MENAMBAHKAN:
1. Siapkan file gambar dengan nama yang sesuai
2. Copy file ke folder public/ ini
3. Restart development server (npm start)
4. Refresh browser (Ctrl+F5)

Untuk panduan lengkap, baca: IMAGE_GUIDE.md
